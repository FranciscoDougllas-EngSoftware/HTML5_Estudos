# Tags HTML5 Mais Utilizadas

## **213 tags HTML5** válidas e relevantes em março de 2025

---

## 1. Estrutura Básica do Documento

### Básico
- `<!DOCTYPE html>` - Declara o documento como HTML5.
- `<html>` - Elemento raiz do documento.
- `<head>` - Contém metadados e informações do documento.
- `<body>` - Contém o conteúdo visível da página.
- `<title>` - Define o título da página (exibido na aba do navegador).
- `<meta>` - Define metadados como codificação (ex.: `<meta charset="UTF-8">`).

### Intermediário
- `<link>` - Conecta recursos externos, como folhas de estilo CSS.
- `<script>` - Inclui ou referencia scripts, como JavaScript.
- `<base>` - Define a URL base para links relativos no documento.

### Avançado
- `<noscript>` - Conteúdo exibido se o JavaScript estiver desativado.
- `<meta name="viewport">` - Configura o comportamento responsivo.

## 2. Texto e Conteúdo

### Básico
- `<h1>` a `<h6>` - Títulos de nível 1 a 6.
- `<p>` - Parágrafo.
- `<br>` - Quebra de linha.
- `<hr>` - Linha horizontal.
- `<strong>` - Texto em negrito com ênfase semântica.
- `<em>` - Texto em itálico com ênfase semântica.
- `<b>` - Texto em negrito (visual apenas).
- `<i>` - Texto em itálico (visual apenas).

### Intermediário
- `<span>` - Container genérico para texto ou elementos inline.
- `<div>` - Container genérico para blocos de conteúdo.
- `<pre>` - Texto pré-formatado (mantém espaços e quebras).
- `<blockquote>` - Citação em bloco.
- `<q>` - Citação inline.
- `<cite>` - Referência a uma fonte ou obra.
- `<code>` - Código de computador.
- `<kbd>` - Entrada de teclado.
- `<samp>` - Saída de exemplo de um programa.
- `<ruby>` - Anotações de texto (usado em idiomas como japonês).
- `<rt>` - Texto de anotação para `<ruby>`.
- `<rp>` - Parênteses para navegadores sem suporte a `<ruby>`.

### Avançado
- `<abbr>` - Abreviação ou acrônimo.
- `<address>` - Informações de contato.
- `<del>` - Texto deletado.
- `<ins>` - Texto inserido.
- `<mark>` - Texto destacado.
- `<sub>` - Texto subscrito.
- `<sup>` - Texto sobrescrito.
- `<small>` - Texto pequeno (notas ou disclaimers).
- `<var>` - Variável em programação ou matemática.
- `<data>` - Dados legíveis por máquina com valor visível.
- `<time>` - Representa uma data ou hora.
- `<bdi>` - Isolamento de direção de texto bidirecional.
- `<bdo>` - Sobrescreve a direção do texto (ex.: direita para esquerda).
- `<wbr>` - Sugere ponto de quebra de palavra.

## 11. MathML (Matemática no HTML)

### Avançado
- `<math>` - Container principal para MathML.
- `<mi>` - Identificador (ex.: variáveis como "x").
- `<mn>` - Número (ex.: "42").
- `<mo>` - Operador (ex.: "+", "−").
- `<ms>` - String literal (ex.: texto entre aspas).
- `<mtext>` - Texto puro em MathML.
- `<mrow>` - Agrupa elementos horizontalmente.
- `<mfrac>` - Fração.
- `<msqrt>` - Raiz quadrada.
- `<mroot>` - Raiz de índice específico.
- `<msub>` - Subscrito.
- `<msup>` - Sobrescrito.
- `<msubsup>` - Subscrito e sobrescrito combinados.
- `<mtable>` - Tabela ou matriz.
- `<mtr>` - Linha em uma tabela MathML.
- `<mtd>` - Célula em uma tabela MathML.
- `<mfenced>` - Adiciona delimitadores (ex.: parênteses) automaticamente.
- `<mpadded>` - Ajusta espaçamento interno.
- `<mspace>` - Espaço ajustável.
- `<menclose>` - Envolve conteúdo com notações (ex.: caixas).
- `<mphantom>` - Espaço invisível para alinhamento.
- `<munder>` - Elemento abaixo de outro.
- `<mover>` - Elemento acima de outro.
- `<munderover>` - Elemento acima e abaixo (ex.: somatórios).
- `<mmultiscripts>` - Scripts múltiplos (ex.: tensores).
